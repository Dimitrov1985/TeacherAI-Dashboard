# 📋 Массовый импорт учеников

## Как это работает

Массовый импорт позволяет добавить сразу весь класс учеников двумя способами:

### 1️⃣ Загрузка фото списка (с AI распознаванием)

**Как использовать:**
1. Откройте вкладку **Students**
2. Нажмите **"Массовый импорт"**
3. Укажите класс (например, "10А")
4. Выберите **"Загрузить фото"**
5. Сфотографируйте или загрузите список класса
6. Нажмите **"Распознать текст с AI"**
7. Claude Vision автоматически распознает имена
8. Проверьте и отредактируйте результат
9. Нажмите **"Импортировать всех"**

**Что можно загружать:**
- 📸 Фото списка класса с телефона
- 📄 Скриншот документа
- 📋 Сканы журналов
- 🖼️ Любое изображение с текстом ФИО

**Форматы изображений:** PNG, JPG, JPEG (до 10MB)

### 2️⃣ Вставка текста (без AI)

**Как использовать:**
1. Откройте вкладку **Students**
2. Нажмите **"Массовый импорт"**
3. Укажите класс (например, "9Б")
4. Выберите **"Вставить текст"**
5. Скопируйте список учеников
6. Вставьте в текстовое поле
7. Нажмите **"Обработать список"**
8. Проверьте и отредактируйте результат
9. Нажмите **"Импортировать всех"**

**Поддерживаемые форматы:**

```
Иванов Иван Иванович
Петров Петр Петрович
Сидорова Анна Сергеевна
```

или с нумерацией:

```
1. Иванов Иван Иванович
2. Петров Петр Петрович
3. Сидорова Анна Сергеевна
```

или через номера со скобками:

```
1) Иванов Иван
2) Петрова Анна Сергеевна
3) Сидоров Петр
```

**Правила:**
- Каждый ученик на **новой строке**
- Формат: **Фамилия Имя [Отчество]**
- Отчество необязательно
- Нумерация автоматически удаляется

## Редактирование перед импортом

Перед финальным импортом вы можете:
- ✏️ Отредактировать любое ФИО
- ❌ Удалить ученика из списка (если распознано лишнее)
- ✅ Проверить правильность данных

## После импорта

Все ученики автоматически:
- ✅ Добавляются в указанный класс
- ✅ Появляются в списке Students
- ✅ Обновляют счётчик на главной странице
- ✅ Доступны для добавления оценок и посещаемости

## Дополнительная информация

После массового импорта вы можете:
1. Открыть карточку любого ученика
2. Добавить контакты, фото, заметки
3. Начать вести журнал оценок
4. Отмечать посещаемость

## Примеры использования

### Пример 1: Начало учебного года
Получите список класса от администрации → Сфотографируйте → Импортируйте за 30 секунд

### Пример 2: Новый класс
Скопируйте список из Excel/Word → Вставьте в текстовое поле → Готово

### Пример 3: Перевод учеников
Загрузите фото старого журнала → AI распознает → Добавьте всех одной кнопкой

## Технологии

- 🤖 **Claude 3.5 Sonnet** с Vision API для распознавания текста
- 🧠 Умное определение ФИО из произвольного текста
- 🔧 Ручная корректировка перед импортом
- 💾 Автоматическое сохранение в localStorage
