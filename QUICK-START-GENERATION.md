# 🚀 Быстрый старт - Генерация планов по фото

## ✅ Всё готово! Просто запустите:

```bash
vercel dev
```

## 📸 Как использовать:

1. **Откройте Dashboard** → Кликните на урок в расписании
2. **Вкладка Materials** → Кнопка "Сгенерировать план урока"
3. **Загрузите фото** страницы учебника (PNG/JPG)
4. **Нажмите Generate** → ⏳ Подождите 5-15 секунд
5. **✅ Готово!** План появится в материалах

## 🎯 Что AI создаёт:

- 📝 Название урока
- 🎯 Цели обучения (3-5)
- 🗂️ Флэшкарты (5+)
- 🎨 Активности для класса
- 📦 Список материалов

## 💡 Советы:

✅ Чёткое фото с хорошим освещением  
✅ Весь текст читабельный  
✅ Без бликов  
❌ Не размытое  
❌ Не обрезанное  

## 🔑 API уже настроен!

Ваш `.env` файл содержит Google API ключ.  
Модель: **Gemini 2.0 Flash** (бесплатная!)

Лимиты:
- 15 запросов/минуту
- 1,500 запросов/день
- 1,000,000 запросов/месяц

## 📥 После генерации:

- **Просмотр** - клик на материал
- **Скачать Word** - кнопка скачать
- **Привязать к уроку** - автоматически
