# 🔧 Быстрое исправление ошибки 500

## ❌ Проблема: 
```
ANTHROPIC_API_KEY=your-anthropic-api-key-here
```
Ключ не настроен!

## ✅ Решение:

### Шаг 1: Получите API ключ

1. Откройте: https://console.anthropic.com/settings/keys
2. Войдите в аккаунт (или зарегистрируйтесь)
3. Нажмите "Create Key"
4. Скопируйте ключ (начинается с `sk-ant-`)

### Шаг 2: Откройте .env файл

Найдите строку:
```
ANTHROPIC_API_KEY=your-anthropic-api-key-here
```

### Шаг 3: Замените на ваш ключ

```
ANTHROPIC_API_KEY=sk-ant-api03-ваш_реальный_ключ_здесь
```

### Шаг 4: Сохраните файл

### Шаг 5: Перезапустите сервер

В терминале:
```bash
# Ctrl+C (остановить)
npm run dev:api
```

### Шаг 6: Попробуйте снова

Откройте http://localhost:3000 и попробуйте генерацию!
