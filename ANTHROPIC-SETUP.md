# 🤖 Настройка Anthropic Claude для генерации планов уроков

## 🔑 Получение API ключа

### Шаг 1: Создайте аккаунт Anthropic

1. Перейдите: **https://console.anthropic.com/**
2. Нажмите **"Sign Up"** или **"Log In"**
3. Зарегистрируйтесь через email или Google

### Шаг 2: Создайте API ключ

1. Откройте: **https://console.anthropic.com/settings/keys**
2. Нажмите **"Create Key"**
3. Дайте название (например: "TeacherHub")
4. Скопируйте ключ (начинается с `sk-ant-...`)

⚠️ **ВАЖНО:** Сохраните ключ сразу - он больше не будет показан!

### Шаг 3: Добавьте ключ в проект

Откройте файл `.env` и вставьте ваш ключ:

```env
ANTHROPIC_API_KEY=sk-ant-api03-...ваш_ключ_здесь
```

### Шаг 4: Перезапустите сервер

```bash
# Остановите текущий сервер (Ctrl+C)
# Затем запустите снова:
vercel dev
```

**Модель:** Claude 3.5 Sonnet (с поддержкой изображений)

**Пример на $5:**

## 🔧 Troubleshooting

### ❌ "ANTHROPIC_API_KEY is not configured"

**Решение:**
1. Проверьте файл `.env`
2. Убедитесь что ключ начинается с `sk-ant-`
3. Перезапустите `vercel dev`

### ❌ "HTTP error: 401"

**Решение:**
- Ключ недействителен
- Создайте новый ключ в консоли Anthropic

### ❌ "HTTP error: 429"

**Решение:**
- Превышен лимит запросов
- Подождите минуту или пополните баланс

### ❌ "Failed to generate lesson plan"

**Решение:**
1. Проверьте интернет-соединение
2. Убедитесь что фото в правильном формате
3. Уменьшите размер фото если > 5MB
4. Проверьте баланс в консоли Anthropic

---

## 💳 Пополнение баланса

1. Откройте: **https://console.anthropic.com/settings/billing**
2. Нажмите **"Add Credits"**
3. Выберите сумму (минимум $5)
4. Оплатите картой

**Рекомендация:**
- $5 = ~250-500 планов уроков
- $10 = ~500-1000 планов уроков
- $20 = ~1000-2000 планов уроков
