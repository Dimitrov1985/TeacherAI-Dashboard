## [Unreleased] - 2026-06-26

### Added
- ✨ **Экспорт планов уроков в формат Word (.docx)**
  - Новая библиотека `docx` для генерации Word документов
  - Новая библиотека `file-saver` для скачивания файлов
  - Утилита `exportToWord.ts` для конвертации планов в Word
  - Кнопка "Скачать Word" в компоненте просмотра плана урока
  - Иконка скачивания в списке материалов для быстрого экспорта
  - Полное руководство по использованию (EXPORT_GUIDE.md)

### Changed
- 📝 Обновлён README с описанием новой функциональности
- 🎨 Улучшен UI компонента `LessonPlanViewer` с новой кнопкой экспорта
- 🎨 Улучшен UI компонента `LessonDetailModal` с иконкой скачивания

### Technical Details
- Документ содержит:
  - Заголовок с названием урока
  - Раздел "Цели урока" со списком целей
  - Раздел "Классные активности" с маркированным списком
  - Раздел "Необходимые материалы" с маркированным списком
  - Раздел "Карточки для запоминания" с таблицей вопрос/ответ
  - Дату создания документа
- Поддержка кириллицы (UTF-8)
- Автоматическая генерация имени файла
- Полная совместимость с MS Word, Google Docs, LibreOffice

### Dependencies
- Added: `docx@^8.5.0`
- Added: `file-saver@^2.0.5`
- Added: `@types/file-saver@^2.0.7`
