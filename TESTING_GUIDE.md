# 🧪 Инструкция по тестированию

## Быстрый тест тайского формата

### Шаг 1: Откройте приложение
Перейдите на http://localhost:5173 и войдите в систему

### Шаг 2: Перейдите на вкладку Students
Нажмите на иконку "Students" в левой панели

### Шаг 3: Массовый импорт через текст

1. Нажмите кнопку **"Массовый импорт"**
2. Укажите класс: **"ม.4/1"** (или любой другой)
3. Выберите **"Вставить текст"**
4. Скопируйте содержимое файла `test_thai_import.txt`:

```
1 67001 สมชาย ใจดี
2 67002 สมหญิง รักดี
3 67003 ประเสริฐ ดี
4 67004 วิชัย สุข
5 67005 มานี แก้ว
6 67006 สุดา จันทร์
7 67007 นิรันดร์ ศรี
8 67008 ปิยะ วงศ์
9 67009 รัตน์ ทอง
10 67010 ศักดิ์ ดา
```

5. Вставьте в текстовое поле
6. Нажмите **"Обработать список"**
7. Проверьте результат:
   - ✅ ID должны быть: 67001, 67002, 67003...
   - ✅ Имена должны быть на тайском
   - ✅ Фамилии должны быть на тайском
8. Нажмите **"Импортировать всех"**

### Шаг 4: Проверка результата

1. ✅ Должны появиться 10 карточек учеников
2. ✅ Каждая карточка показывает:
   - Аватар с инициалами (тайские буквы)
   - Фамилию и имя на тайском
   - Класс "ม.4/1"
   - ID номер (например, "ID: 67001")

### Шаг 5: Детальный просмотр

1. Нажмите на любую карточку ученика
2. ✅ Откроется детальный просмотр
3. ✅ Во вкладке "Информация" должен быть виден "ID No. (ลำดับ)"
4. ✅ Попробуйте добавить оценку
5. ✅ Попробуйте отметить посещаемость

### Шаг 6: Тест удаления класса

1. В фильтре выберите класс "ม.4/1"
2. ✅ Должна появиться красная кнопка **"Удалить класс"**
3. Нажмите на неё
4. ✅ Должно появиться подтверждение с количеством учеников
5. Подтвердите удаление
6. ✅ Все ученики класса должны исчезнуть

## Тест AI распознавания (через фото)

1. Создайте документ со списком учеников
2. Сфотографируйте или сделайте скриншот
3. Загрузите через **"Массовый импорт"** → **"Загрузить фото"**
4. Нажмите **"Распознать текст с AI"**
5. Проверьте результат

**Примечание:** Для работы AI нужен `ANTHROPIC_API_KEY` в `.env.local`

## Известные проблемы

1. **AI распознавание требует интернет** - работает только при наличии API ключа
2. **Тайский шрифт** - убедитесь, что в браузере установлена поддержка тайского языка
3. **localStorage** - данные хранятся локально, очистка кэша удалит всё
