# Руководство по экспорту планов уроков в Word

## Обзор функциональности

Теперь вы можете скачивать сгенерированные планы уроков в формате Microsoft Word (.docx)!

## Как использовать

### Способ 1: Из просмотра плана урока

1. Откройте любой урок в расписании
2. Перейдите на вкладку "Materials"
3. Нажмите кнопку "Сгенерировать план урока"
4. Загрузите фото страницы учебника
5. Дождитесь генерации плана
6. Откройте созданный план (нажмите на него в списке материалов)
7. В открывшемся окне нажмите кнопку **"Скачать Word"**
8. Документ автоматически скачается на ваш компьютер

### Способ 2: Прямо из списка материалов

1. Откройте любой урок с сгенерированным планом
2. В списке материалов найдите план урока (отмечен иконкой 🗂️)
3. Нажмите на иконку скачивания (📥) справа от плана
4. Документ автоматически скачается

## Что включает Word документ

Экспортированный документ содержит:

- **Заголовок урока** - название плана
- **Цели урока** - список образовательных целей
- **Классные активности** - предлагаемые виды деятельности
- **Необходимые материалы** - список требуемых ресурсов
- **Карточки для запоминания** - таблица с вопросами и ответами
- **Дата создания** - автоматически добавляется в конец документа

## Формат документа

- Все заголовки оформлены в едином стиле
- Списки автоматически маркированы
- Карточки представлены в виде удобной таблицы
- Документ полностью редактируемый после скачивания

## Технические детали

- **Формат**: .docx (совместим с Microsoft Word, Google Docs, LibreOffice)
- **Название файла**: автоматически генерируется из названия плана + timestamp
- **Кодировка**: UTF-8 (поддержка кириллицы)
- **Размер**: обычно 20-50 KB

## Возможные проблемы и решения

### Документ не скачивается

1. Проверьте, что браузер не блокирует всплывающие окна
2. Убедитесь, что у браузера есть права на скачивание файлов
3. Попробуйте другой браузер (Chrome, Firefox, Edge)

### Документ скачивается, но не открывается

1. Убедитесь, что у вас установлен Microsoft Word, Google Docs или LibreOffice
2. Проверьте, что файл полностью скачался (размер больше 0 KB)
3. Попробуйте открыть через Google Docs онлайн

### Кириллица отображается неправильно

- Это не должно происходить, так как используется UTF-8
- Если проблема возникла, откройте документ в Google Docs и пересохраните
