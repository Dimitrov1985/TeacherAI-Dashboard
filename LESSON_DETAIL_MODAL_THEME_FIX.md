## 💡 Технические детали

### Метод исправления: Глобальная замена

Вместо ручного редактирования каждого элемента использовал **replace_all: true**:

```tsx
// Пример
Edit({
  file_path: 'LessonDetailModal.tsx',
  old_string: 'text-[#1D3557]',
  new_string: 'text-[var(--text-primary)]',
  replace_all: true  // Заменить ВСЕ вхождения
})
```

**Преимущества:**
- ✅ Быстро (1 команда вместо 10+)
- ✅ Консистентно (все одинаково)
- ✅ Безопасно (все вхождения заменены)
