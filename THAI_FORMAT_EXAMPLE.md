# 🇹🇭 Тайский формат списка учеников

## Поддерживаемые форматы

Система теперь полностью поддерживает тайский формат со следующими полями:

| ลำดับ | ID No. | ชื่อ - สกุล | หมายเหตุ |
|-------|--------|-------------|----------|
| 1     | 12345  | สมชาย ใจดี  | -        |
| 2     | 12346  | สมหญิง รักดี | -        |
| 3     | 12347  | ประเสริฐ ดี  | -        |

Где:
- **ลำดับ** - порядковый номер (игнорируется)
- **ID No.** - номер студента (сохраняется как `studentId`)
- **ชื่อ - สกุล** - имя и фамилия
- **หมายเหตุ** - примечания (можно добавить в Notes)

## Примеры импорта

### Вариант 1: Через текст

Скопируйте и вставьте в поле "Массовый импорт" → "Вставить текст":

```
1 12345 สมชาย ใจดี
2 12346 สมหญิง รักดี
3 12347 ประเสริฐ ดี
4 12348 วิชัย สุข
5 12349 มานี แก้ว
```

### Вариант 2: Через фото

1. Сфотографируйте таблицу со списком класса
2. Загрузите в "Массовый импорт" → "Загрузить фото"
3. AI автоматически распознает тайский текст
4. Проверьте и импортируйте

## Смешанные форматы

Система также поддерживает:

### Русский формат:
```
1. Иванов Иван Иванович
2. Петрова Анна Сергеевна
```

### Английский формат с ID:
```
1 12345 John Smith
2 12346 Jane Doe
```

### Тайский формат с табуляцией:
```
1	12345	สมชาย	ใจดี
2	12346	สมหญิง	รักดี
```

## Поля в системе

После импорта для каждого ученика доступны:

- ✅ **ID No. (ลำดับ)** - номер студента
- ✅ **Фамилия / สกุล** - фамилия
- ✅ **Имя / ชื่อ** - имя  
- ✅ **Отчество** - отчество (опционально)
- ✅ **Класс** - класс ученика
- ✅ **Заметки / หมายเหตุ** - примечания учителя
- ✅ **Контакты** - email, телефон
- ✅ **Родители** - контакты родителей

## Журнал оценок

После добавления учеников можно:
- ✅ Выставлять оценки по предметам
- ✅ Отмечать посещаемость
- ✅ Просматривать статистику
- ✅ Добавлять заметки

## Технические детали

**AI распознавание:**
- Модель: Claude 3.5 Sonnet с Vision API
- Поддержка: тайский, русский, английский, смешанные
- Точность: >95% при хорошем качестве фото

**Ручной парсинг:**
- Автоматическое определение формата
- Поддержка разделителей: пробел, табуляция
- Умная обработка ID и порядковых номеров

## Советы

1. **Для лучшего распознавания фото:**
   - Хорошее освещение
   - Четкий фокус
   - Прямой угол съемки
   - Контрастный текст

2. **Для текстового импорта:**
   - Каждый ученик на новой строке
   - Одинаковый формат для всех строк
   - Разделитель: пробел или табуляция

3. **Редактирование:**
   - Всегда проверяйте результат перед импортом
   - Можно редактировать любое поле
   - Можно удалить лишние строки

## Пример реального использования

**Исходная таблица:**
```
ลำดับ    ID No.    ชื่อ - สกุล        หมายเหตุ
1        67001     สมชาย ใจดี         
2        67002     สมหญิง รักดี       เรียนดี
3        67003     ประเสริฐ ดี        
```

**После импорта в систему:**
- Student ID: 67001, 67002, 67003
- Имена и фамилии на тайском
- Класс указан при импорте
- Заметки можно добавить после

---

**Поддержка**: Все функции работают одинаково для тайского, русского и английского языков! 🌏
