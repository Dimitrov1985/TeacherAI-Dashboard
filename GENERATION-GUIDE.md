# 📸 Генерация планов уроков по фото - Руководство

## ✅ Всё уже настроено!

Ваш проект полностью готов к генерации планов уроков по фотографиям учебников.

---

## 🚀 Как использовать:

### **1. Запустите проект локально с API:**

```bash
vercel dev
```

**Важно!** Используйте `vercel dev`, а не `npm run dev`, чтобы API endpoints работали.

### **2. Откройте приложение:**

Откроется на `http://localhost:3000` (или другой порт)

### **3. Создайте план урока по фото:**

#### **Вариант А - Из расписания:**

1. Откройте Dashboard
2. Кликните на любой урок в Weekly Schedule
3. В модальном окне → вкладка "Materials"
4. Кнопка **"Сгенерировать план урока"**
5. **Загрузите фото** страницы учебника (PNG, JPG)
6. Кнопка **"Generate"**
7. ⏳ AI анализирует фото (5-15 секунд)
8. ✅ Готовый план урока появится в Materials

#### **Вариант Б - Из "Мои уроки":**

1. Откройте страницу "My lessons"
2. Кнопка **"✨ Сгенерировать план урока"**
3. Загрузите фото → Generate
4. ✅ План сохранится в библиотеку

---

## 🤖 Что делает AI:

**Анализирует фото учебника и создаёт:**

1. **📝 Название урока** (title)
2. **🎯 Цели обучения** (3-5 objectives)
3. **🗂️ Флэшкарты** (минимум 5 карточек с терминами/понятиями)
4. **🎨 Активности для класса** (2-3 практических задания)
5. **📦 Необходимые материалы** (список материалов для урока)

## 🔧 Troubleshooting:

### **Ошибка: "GOOGLE_API_KEY is not configured"**

➡️ Решение: Проверьте `.env` файл и перезапустите `vercel dev`

### **Ошибка: "Failed to generate lesson plan"**

➡️ Проверьте:
1. Интернет-соединение
2. API key валидный
3. Фото в формате PNG/JPG
4. Размер фото не слишком большой (< 10MB)

### **Ошибка: "Method not allowed"**

➡️ Используйте `vercel dev` вместо `npm run dev`

### **Долго генерирует (> 30 секунд)**

➡️ Проверьте:
1. Скорость интернета
2. Размер фото (уменьшите если > 5MB)
3. Попробуйте перезапустить `vercel dev`
